FACEBOOK INFER

Out of the many projects, Facebook Infer is one of the more known successes of the PPLV department. What exactly is Facebook Infer? It is a collaboration between the PPLV department (mainly Professor Peter O'Hearn)[1] and Facebook where a program was created to identify bugs within source code before the code is even run![2] You may be thinking that this may seem pointless because identifying bugs can be simple through a little searching of one’s code. But, as programs get bigger, so does the potential for bugs, and it becomes ever more difficult to identify them manually. Hence, the Facebook Infer is a very important and useful analysis tool for high-level programmers to help prevent crashes or poor performance.
Visit the Official Facebook Infer site to learn more![3]

PRODUCTIVE SECURITY

Another impact of the PPLV department was the development of the “Productive Security”. This project is a framework where it takes as an input a company’s / organization’s workload and risk perception and tells the programmers what sort of security measures is most efficient for said company. A very simple and basic example is that it would tell a company what sort of password system to use, such as an eight long character system or a four pin code digit system. This can be extremely helpful for security purposes as not all organizations hold the same type of information / data structure and thus will require different measures of security. As obvious as it sounds, this project was aimed to higher the security compliance of organizations around the globe. [3]

Gnosis

One other example of a huge impact of the PPLV department was providing the foundation for the development and creation of Gnosis. This program is a predictive marketing system, which means that it can be used in different sections of the tertiary sector. One of its many operations is predicting media and celebrity gossip. How can this be beneficial? Well, it may not seem important to you, but powerful celebrities can massively impact stock markets and financial businesses within a few clicks. One of the biggest examples is the $5.7 Billion Dollar downfall of Amazon on stock markets after President Donald Trump’s tweet - “Amazon is doing great damage to tax paying retailers. Towns, cities and states throughout the US are being hurt - many jobs being lost!” [4]
